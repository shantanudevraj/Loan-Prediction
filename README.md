# Loan-Prediction

So, this is basically a classification problem and  below are the steps i used to predict the yes/no for getting loan or not with 80 % accuracy.
steps used 
1. Data Preprocessing 
2. Label Encoding and One hot encoding
3.Alorithm used : KNN
